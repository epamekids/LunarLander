## Формулы

### Зависимость ускорения свободного падения от высоты над поверхностью тела
`g = G * M / (R + h)^2`

`g (m/s^2)` - ускорение свободного падения на высоте `h (m)` над поверхностью тела радиусом `R (m)` и массой `M (kg)`
