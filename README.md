# Lunar Lander
# Посадка на Луну

## Константы
### Луна
**Масса** `Mm = 7.3477e22 kg`

**Радиус** `Rm = 1737.10 km`

## Формулы

### Зависимость ускорения свобоного падения от высоты над поверхностью тела
`g = G * M / (R + h)`
`g` m/s^2 - ускорени
`G`- гравитационная постоянная

`M` - масса тела
`R` - радиус тела
`h` - высота над поверхностью
